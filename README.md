# Sobre o Projeto

O EXOBIOS é uma Landing Page desenvolvida para apresentar uma solução tecnológica voltada ao cultivo de alimentos em ambientes extremos, como bases espaciais na Lua e em Marte.

A proposta do projeto é demonstrar como sistemas inteligentes podem monitorar e controlar fatores ambientais essenciais para o desenvolvimento de culturas agrícolas, garantindo condições adequadas mesmo em locais onde a agricultura tradicional não é possível.

A página foi desenvolvida utilizando HTML, CSS e JavaScript, aplicando conceitos de design responsivo, interatividade e boas práticas de desenvolvimento web.

---

# Problema

Missões espaciais de longa duração exigem autonomia na produção de alimentos.

Ambientes como a Lua e Marte apresentam condições hostis ao cultivo, incluindo variações extremas de temperatura, ausência de solo fértil e dificuldades no controle de umidade e luminosidade.

Esses fatores tornam necessário o desenvolvimento de sistemas automatizados capazes de monitorar e ajustar as condições ambientais de cultivo.

---

# Solução Proposta

O EXOBIOS apresenta uma estufa inteligente capaz de monitorar:

* Temperatura;
* Umidade do ar;
* Luminosidade.

O sistema permite selecionar diferentes culturas agrícolas e ajustar automaticamente os parâmetros ideais para cada planta, simulando uma solução para produção de alimentos em futuras colônias espaciais.
